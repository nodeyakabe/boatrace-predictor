## 調査結果: 乖離の原因

### 1. 検証期間の不一致

**残タスク一覧の記載**:
- 検証期間: 2025年1月〜12月
- オッズ付きレース数: **13,413レース**

**実際のデータベース**:
- 2025年全期間: 10,705レース（オッズ&予測付き）
- 2025年1-8月: 9,959レース
- 2025年1-10月: 10,350レース
- 2025年1-12月(12/5まで): 10,705レース

**結論**: **13,413レースというデータセットは存在しない**

### 2. ROI算出方法の不明確さ

複数の可能性が考えられます:

#### 可能性A: 的中レースの平均配当をROIとして記載
- 例: イン強会場条件の的中9件の平均配当 38.2倍 → ROI 38.2%と解釈
- しかし、残タスク記載のROI 120.6%とは一致しない

#### 可能性B: 理論値計算
- サンプルレースから理論的期待値を計算
- 実際の全レース購入ベースのROIとは異なる

#### 可能性C: 部分期間での高パフォーマンス
- 2025年の一部期間のみで計算
- ただし、どの期間でも120%超のROIは確認できず

### 3. 実装条件の相違

BetTargetEvaluatorの実装条件:
```python
# 信頼度C（実装済み）
'C': [
    {'method': '新方式', 'odds_min': 5, 'odds_max': 15, 'c1_rank': ['A1'], ...},
    {'method': '新方式', 'odds_min': 15, 'odds_max': 50, 'c1_rank': ['A1'], ...},
]

# 信頼度D（実装済み）
'D': [
    {'method': '両方式', 'odds_min': 10, 'odds_max': 200, 'c1_rank': ['B1'],
     'venue_codes': [24, 19, 18], ...},  # イン強会場
    {'method': '新方式', 'odds_min': 25, 'odds_max': 50, 'c1_rank': ['A1'], ...},
    {'method': '従来', 'odds_min': 20, 'odds_max': 50, 'c1_rank': ['A1'], ...},
]
```

残タスク一覧の条件とは微妙に異なる点:
- 残タスク: 「C × 従来」
- 実装: 「C × 新方式」

### 4. 過去データでの検証不可

- **予測データは2025年のみ存在**
- 2024年以前のデータでは検証不可能
- 過去の高ROI期間を特定できない

## 結論

### 期待ROI（127%-251%）は信頼できない

以下の理由により、残タスク一覧に記載された期待ROI値は**実データに基づいていない可能性が高い**:

1. ✗ 記載された「13,413レース」のデータセットが存在しない
2. ✗ 実測ROIが全条件で0.5%-2.2%と大幅に低い（差分118%-251%）
3. ✗ 2025年のどの期間・月でも120%超のROIは確認できない
4. ✗ 過去データ（2024年以前）での検証が不可能（予測データなし）
5. ✗ ROI算出方法の記載がなく、再現不可能

### 実運用での期待値

**現実的な期待ROI**: **0.5% - 2.0%**

- 的中率: 3-4%
- 平均配当: 10-40倍
- 月間購入: 約70-90レース
- 月間的中: 約2-3レース
- 月間収支: ほぼトントン〜微損
