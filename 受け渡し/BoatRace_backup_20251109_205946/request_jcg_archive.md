# 海上保安庁 潮位データアーカイブ請求

## 概要
海上保安庁 海洋情報部に対して、2015-2021年の潮位観測データのアーカイブ提供を依頼

---

## 依頼先

### 海上保安庁 海洋情報部
- **公式サイト**: https://www1.kaiho.mlit.go.jp/
- **潮汐情報**: https://www1.kaiho.mlit.go.jp/KANKYO/TIDE/
- **問い合わせ**: https://www1.kaiho.mlit.go.jp/JODC/info/contact.html

## 依頼文テンプレート

### 件名
```
潮位観測データの提供依頼（研究・分析目的）
```

### 本文
```
海上保安庁 海洋情報部 御中

お世話になっております。

競艇レース予想システムの研究開発を行っている者です。
海水面の潮位変動がレース結果に与える影響を分析する目的で、
下記期間の潮位観測データの提供をお願いできますでしょうか。


【希望データ】

1. データ期間
   2015年11月1日 ～ 2021年12月31日

2. 観測地点（以下のいずれか、または最寄りの観測点）
   - 高松（丸亀競艇場最寄り）
   - 宇野（児島競艇場最寄り）
   - 広島（宮島競艇場最寄り）
   - 徳山（徳山競艇場最寄り）
   - 若松（若松競艇場最寄り）
   - 博多（福岡競艇場最寄り）
   - 長崎（大村競艇場最寄り）

3. データ項目
   - 観測日時
   - 潮位（cm）
   - （可能であれば）気圧、気温

4. データ頻度
   - 毎時値、または10分値（可能な範囲で）
   - 最低限、満潮・干潮時刻と潮位

5. データ形式
   - CSV形式、またはテキスト形式


【利用目的】
- 競艇レース予想システムの機械学習モデルの特徴量として使用
- 非営利の研究・分析目的
- データの再配布は行いません


【背景】
気象庁のNEAR-GOOS RDMDBでは2022年11月以降のデータしか
公開されておらず、過去データの入手ができない状況です。
海上保安庁様にアーカイブデータが保管されている可能性を考え、
ご連絡させていただきました。


お手数をおかけいたしますが、ご検討のほどよろしくお願いいたします。


【連絡先】
氏名: [氏名を記入]
メールアドレス: [メールアドレスを記入]
電話番号: [電話番号を記入]
```

## 依頼時の注意点

### 1. 正式な依頼であることを示す
- 実名で依頼
- 連絡先（メール・電話）を明記
- 所属（大学、企業など）があれば記載

### 2. 利用目的を明確にする
- 研究・分析目的であることを強調
- 営利目的でないことを明示
- データの再配布をしないことを約束

### 3. データ形式の柔軟性を示す
- 「可能な範囲で」を強調
- 複数の選択肢を提示
- 先方の負担を考慮

### 4. 感謝の意を示す
- 丁寧な言葉遣い
- 公的機関への敬意

## 期待される結果

### ケース1: データ提供可能（理想）
- アーカイブデータをCSV/TXT形式で提供
- 2-4週間程度で受領
- **費用**: 無料～数千円程度（コピー代など）

### ケース2: 提供不可だが代替案の提示
- 「潮汐表（PDF）を参照してください」
- 「○○研究機関にデータがあります」
- 有料データサービスの紹介

### ケース3: 提供不可
- データが保管されていない
- 提供基準を満たさない
- → 他の方法（PyTides推定など）を使用

## 並行して実施すべきこと

### 1. PyTides推定を先行実施
```bash
# すぐに実装できる方法で先に進める
python estimate_tide_pytides.py --start 2015-11-01 --end 2021-12-31
```

### 2. 2022年以降のデータを完全活用
```bash
# 既存データの紐付けを完了
python link_tide_to_races.py --start 2022-11-01 --end 2025-10-31
```

### 3. 潮位なしでモデル構築開始
- 2015-2021年: 潮位推定値
- 2022年以降: 実測値
- → ハイブリッドアプローチ

## まとめ

### 推奨アプローチ
1. **即座**: PyTides推定で2015-2021年をカバー
2. **並行**: 海上保安庁にアーカイブ請求
3. **中期**: データ受領できれば推定値を実測値に置き換え

### メリット
- すぐに機械学習を開始できる
- 長期的により高品質なデータを入手できる可能性
- 低コスト（無料～数千円）

### デメリット
- 海上保安庁からの提供は不確実
- データ受領まで数週間～数ヶ月かかる可能性
- 依頼・交渉の手間

**結論**: PyTides推定ですぐに開始し、並行して海上保安庁に依頼するのが最適
