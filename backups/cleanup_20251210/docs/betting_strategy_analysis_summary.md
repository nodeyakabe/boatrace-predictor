## 4. 重要な発見

### 4.1 級別フィルターの必要性
- 2連単は級別フィルターなしでは**全信頼度で赤字**
- A1級に絞ることで初めて黒字（D×A1: ROI 106.7%）
- 理由: A1級レースは払戻が高く、予測精度も維持できる

### 4.2 信頼度別の特徴
| 信頼度 | 特徴 | 2連単向き？ |
|--------|------|-------------|
| A | サンプル極少 | 判断不可 |
| B | 的中率高（21%）だが払戻低 | A2級のみ微黒字 |
| C | 件数多いが払戻低 | 全て赤字 |
| D | 的中率低めだが払戻高め | **A1級で黒字** |

### 4.3 複合戦略の効果
- **メリット**: 月間的中数が2.9件→14.9件（5倍）に増加
- **デメリット**: ROIが122.9%→117.9%に低下、黒字月も減少
- 2連単は「的中実感を増やす補助戦略」として有効
