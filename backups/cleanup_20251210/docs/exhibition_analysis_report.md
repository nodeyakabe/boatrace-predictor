# 展示ST・展示タイム 詳細分析レポート

**分析日**: 2024年11月27日
**分析期間**: 2024年1月1日〜2024年11月20日

---

## 結論

**展示ST・展示タイムの効果は「本物」だが、交絡因子による過大評価がある**

- コースと強い交絡あり（1コースの選手がST1位になりやすい）
- 級別と中程度の交絡あり（A1選手がST1位になりやすい）
- ただし、同一コース内でも展示ST1位は明確に勝率が高い

---

## 1. 交絡因子の確認

### 1.1 コースとの交絡

**1コースの選手は展示ST1位になりやすい（32.94% vs 平均16.67%）**

| コース | ST1位率 | 平均ST順位 |
|--------|--------|-----------|
| 1コース | **32.94%** | 2.95 |
| 2コース | 15.14% | 3.53 |
| 3コース | 19.28% | 3.23 |
| 4コース | 19.94% | 3.33 |
| 5コース | 16.87% | 3.53 |
| 6コース | 14.90% | 4.03 |

**解釈**: 1コースの選手は展示STで速いタイムを出しやすい。これは1コースに強い選手が配置されることが多いためと思われる。

### 1.2 級別との交絡

**A1選手は展示ST1位になりやすい（23.58% vs B2選手17.05%）**

| 級別 | ST1位率 | 平均ST順位 |
|------|--------|-----------|
| A1 | **23.58%** | 3.23 |
| A2 | 21.43% | 3.32 |
| B1 | 17.77% | 3.53 |
| B2 | 17.05% | 3.81 |

**解釈**: 上位選手ほどスタートが速い傾向がある。展示ST1位の勝率が高いのは、一部は選手の実力差を反映している。

## 2. 同一コース内での展示STの効果（純粋な効果）

**交絡を除去した分析結果**

| コース | ST1位の勝率 | ST2位以下の勝率 | 差分 |
|--------|-----------|--------------|------|
| 1コース | **75.11%** | 50.46% | **+24.65%** |
| 2コース | **21.59%** | 10.74% | +10.85% |
| 3コース | **19.06%** | 8.97% | +10.09% |
| 4コース | **23.63%** | 6.12% | +17.51% |
| 5コース | **11.85%** | 3.78% | +8.07% |
| 6コース | **7.98%** | 1.76% | +6.22% |

**重要な発見**:
- **1コースでの効果が特に大きい（+24.65%）**
- 全コースで展示ST1位の方が勝率が高い
- **これは交絡ではなく、展示STの純粋な効果**

## 7. 改善案への影響

### 現在の設定の妥当性

| 要素 | 現在の重み | 再評価 |
|------|-----------|--------|
| 展示ST | 15% | **妥当**（純粋な効果あり） |
| 展示タイム | 10% | **やや過大**（効果は小さい） |

### 推奨される調整

1. **展示ST（15%）**: 維持
   - 同一コース内でも明確な効果あり
   - 特に1コースと4コースで重要

2. **展示タイム（10% → 5%?）**: 削減を検討
   - 効果は展示STより小さい
   - 1コースでの1位-6位差が15.96%しかない

3. **コース×展示ST の交互作用**: 追加を検討
   - 1コースでの展示ST効果が特に大きい
   - 4コースでも効果大（まくり）

---

## 8. 結論

### 展示STは有効な指標

- 同一コース内で比較しても効果あり（交絡ではない）
- 特に1コースと4コースで予測精度向上に寄与

### 展示タイムは効果が限定的

- コース有利度の影響を超える効果は小さい
- 重み10%は過大評価の可能性

### 推奨アクション

1. 展示STの重み15%は維持
2. 展示タイムの重みを10%→5%に削減を検討
3. 浮いた5%を選手スコアに追加（31%→36%）
