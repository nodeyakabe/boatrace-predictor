### 推奨戦略（1点買い）

```
【信頼度B】
- 方式: 新方式
- 買い目: 1点買い
- 条件: 全レース参加
- 期待回収率: 約120%

【信頼度C】
- 方式: 従来方式
- 買い目: 1点買い
- 条件: 事前オッズ 20〜50倍のレースのみ
- 期待回収率: 約150%

【信頼度D】
- 方式: 新方式
- 買い目: 1点買い
- 条件: 新方式で1-2予測 かつ オッズ20倍以上
- 期待回収率: 約380%（高ボラティリティ）
```

### 5点買いとの比較

| 買い方 | 的中率 | 回収率 | 評価 |
|--------|--------|--------|------|
| 1点買い | 低 | 高 | 推奨 |
| 5点買い | 高 | 低 | 非推奨 |

**理由**: 追加4点で拾えるのは低オッズ（5-15倍）が多く、投資効率が悪化する

## 5. 重要な発見と課題

### 発見事項

1. **オッズフィルタの重要性**
   - オッズデータがある11月は全信頼度で100%超え
   - オッズデータなしの月は80-90%台に低下
   - 適切なオッズ範囲でのフィルタリングが収益の鍵

2. **信頼度Cの最適条件**
   - 従来方式 + オッズ20-50倍で最高効率
   - 新方式より従来方式が有効（理由：Cは混戦レースで複合ルールの適用が難しい）

3. **信頼度Dの活用法**
   - 新方式で1-2予測になるレースが高収益
   - 低的中率だが高配当で回収率が跳ねる
   - 穴狙い戦略として有効

4. **1点買いの優位性**
   - 追加点は低オッズを拾うため効率悪化
   - 的中時の配当を最大化する1点が最適

### 課題

1. **オッズデータの不足**
   - 1月・10月は事前オッズデータがない
   - フィルタ条件が適用できず精度が不明
   - **今後の運用では事前オッズの取得が必須**

2. **検証期間の限界**
   - オッズ付きデータは11月のみ（約1ヶ月分）
   - 長期的な安定性は未検証
   - 季節変動・場ごとの傾向は考慮できていない

3. **サンプル数の問題**
   - 信頼度Dの条件適合（1-2予測 + オッズ20倍以上）は月35件程度
   - 統計的な信頼性には追加データが必要

---

## 6. 今後の方針

### 短期（データ収集）

1. **事前オッズの継続取得**
   - 毎日の予測実行時にオッズも取得・保存
   - 最低3ヶ月分のデータ蓄積を目標

2. **実績の追跡**
   - 推奨戦略での仮想購入記録
   - 回収率の推移モニタリング

### 中期（戦略検証）

1. **オッズデータ蓄積後の再検証**
   - 3ヶ月分データで同条件バックテスト
   - 季節・場別の傾向分析

2. **条件の微調整**
   - オッズ範囲の最適化（例: C は 25-45倍の方が良いか？）
   - 追加条件の探索（1コースランク、スコア差など）

### 長期（運用最適化）

1. **資金管理の導入**
   - 信頼度別の賭け金配分
   - ケリー基準等の適用検討

2. **自動化**
   - 条件に合致するレースの自動抽出
   - 購入推奨の通知システム

---

## 7. 結論

**現時点での最適戦略は確立できたが、検証データが不十分**

- 11月データのみでは戦略の安定性を保証できない
- オッズデータの継続取得が最優先課題
- 3ヶ月以上のオッズ付きデータが蓄積されてから本格運用を推奨

**暫定的な推奨**:
- 信頼度B: 新方式1点買い（全レース）→ 低リスク
- 信頼度C: 従来方式1点買い（オッズ20-50倍）→ 中リスク
- 信頼度D: 新方式1点買い（1-2予測 + オッズ20倍以上）→ 高リスク・高リターン
