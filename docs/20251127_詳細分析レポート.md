def get_dynamic_weights(venue_code: str) -> Dict:
    """会場特性に応じた動的配点"""

        }

## 6. まとめ

### 6.1 主要発見事項

1. **会場特性の影響は非常に大きい**
   - 1コース勝率: 45.4%〜67.1%（22ポイント差）
   - 荒れ率: 33.3%〜54.9%（21ポイント差）

2. **STタイムは予測精度に直結**
   - 平均ST 0.12未満 vs 0.20以上で勝率3倍差
   - 1コースでのST影響が特に大きい（23ポイント差）

3. **展示タイムは有効な指標**
   - 1位 vs 6位で勝率3倍差
   - レース当日の調子を反映

4. **モーターの影響は限定的**
   - 最大でも4%程度の差
   - 会場によって影響度が異なる

5. **前付け傾向選手の把握が重要**
   - 進入予測の信頼度に影響

### 6.2 推奨アクション

1. **動的配点システムの導入**
   - 会場別1コース勝率に応じた配点切替

2. **ST関連スコアの強化**
   - 平均STスコアの重みを10に引き上げ

3. **展示タイムスコアの強化**
   - 展示タイムスコアの重みを10に引き上げ

4. **前付け選手リストの活用**
   - 進入予測時の信頼度調整に使用
