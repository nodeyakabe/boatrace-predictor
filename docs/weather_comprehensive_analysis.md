## エグゼクティブサマリー

風向×風速×会場×級別の複合分析により、以下の重要な発見がありました：

### 🎯 最重要発見

1. **会場依存性が極めて強い**
   - 会場14（三国）西北西風: 1コース勝率 0.0%

3. **風速単独でのルール適用は危険**
   - 会場×風向を考慮しないと誤判定のリスク
   - ただしデータ不足により実装困難

## 4. 複合的な洞察

### 発見1: 会場×風向の重要性

```
同じ暴風(8m+)でも:
  会場14 × 西北西風 → 1コース勝率 0.0%
  会場10 × 西風     → 1コース勝率 50.0%
```

**原因（推測）**:
- 各会場のコースレイアウト（向き）が異なる
- 会場14: 西北西風が向かい風または横風
- 会場10: 西風が追い風または斜め追い風

### 発見2: 暴風時は級別無効化

```
通常時:
  A1級1コース: 63.3%
  B1級1コース: 39.7%
  差: 23.6%

暴風時:
  A1級1コース: 20.0%
  B1級1コース: 20.0%
  差: 0.0% ← 完全に消滅
```

**メカニズム（推測）**:
1. 暴風 → スタート技術が無効化
2. インコースのスタート優位性が失われる
3. 技術差よりも運・枠順が支配的に

### 発見3: 暴風時は4コースが最有利

```
暴風時の勝率:
  1コース: 21.4% (全級別平均)
  4コース: 33.3% (全級別平均)

  B1級4コース: 37.5% ← 最高
```

**メカニズム**:
- 4コースはまくり・まくり差しの起点
- 暴風でインコースが乱れる → 4コースのチャンス拡大
- 技術が低いB1級でも4コースなら勝てる

## 5. 実装への示唆

### 現状の課題

1. **データ不足**
   - 暴風: 15レース → 会場×風向では2-6レース
   - 統計的信頼性が低い

2. **会場別データ不在**
   - 各会場のコースレイアウト情報なし
   - 追い風・向かい風の判定不可

3. **複雑性 vs 効果**
   - 会場×風向×級別のルールは複雑すぎる
   - サンプル数が少なく過学習のリスク

### 推奨アプローチ

#### フェーズ1: シンプルなルール（現在）

**風速のみでルール化**（会場・風向・級別は無視）

```python
if wind_speed >= 8.0:
    if course == 1:
        score -= 20.0  # 全会場平均
    elif course == 4:
        score += 15.0
```

**理由**:
- サンプル数: 15レース（87艇）→ ある程度信頼性あり
- シンプルで過学習リスクが低い
- 平均的には正しい

**リスク**:
- 会場10（西風）のような例外で誤判定
- ただし全体の26%（4/15レース）のみ

#### フェーズ2: 会場別データ収集後

**2025年通年データ収集後に再評価**

目標サンプル数:
- 各会場×風速帯: 50+レース
- 各会場×風向×風速帯: 20+レース

収集後の実装:
```python
# 会場別の風向特性テーブル
VENUE_WIND_EFFECTS = {
    '14': {  # 三国
        '西北西': {'8m+': {'1コース': -30, '4コース': +20}},
        '北北西': {'8m+': {'1コース': -25, '4コース': +15}},
    },
    '10': {  # 桐生
        '西': {'8m+': {'1コース': 0, '4コース': -5}},
    },
    # ... 他の会場
}
```

#### フェーズ3: 級別考慮（オプション）

**暴風時のみ級別補正を無効化**

```python
if wind_speed >= 8.0:
    # 級別による補正を0にする
    racer_skill_bonus = 0
```

**理由**:
- 暴風時は級別差が消滅
- 技術補正が逆効果になる可能性

## 6. 結論

### 重要な発見のまとめ

1. **会場依存性が極めて強い**
   - 同じ風速・風向でも会場で結果が真逆
   - 会場別データなしでは精度の高いルール化は困難

2. **級別は暴風時に無力化**
   - A1級もB1級も1コース勝率20%に収束
   - 技術よりも枠順（特に4コース）が重要に

3. **現時点では風速のみでルール化が妥当**
   - サンプル数の制約
   - 会場別データの不足
   - シンプルさと効果のバランス
