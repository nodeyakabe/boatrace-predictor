## 5. 環境要因減点との関係

### 5.1 現状（減点前）

- 信頼度Bレース数: 5,537
- 全体的中率: 66.82%

### 5.2 環境要因減点適用後の予測

**調整後の分布**（検証済み）:
- B維持: 2,830レース (51.1%) - 的中率73.61%
- B→C降格: 2,472レース (44.7%) - 的中率60.77%
- B→D降格: 235レース (4.2%) - 的中率59.75%

**戦略への影響**:
- **B維持レース**: 戦略B-1/B-2/B-3で積極的に購入
- **B→C降格**: 既存の戦略AのC条件で自動購入
- **B→D降格**: 既存の戦略AのD条件で自動購入

**結論**: 環境要因減点により、信頼度Bの購入対象が以下のように最適化される
1. 環境的に優良なB維持レース → 戦略Bで購入（的中率73.61%）
2. 環境的に不利なC/D降格レース → 戦略Aで購入（的中率59-60%）
