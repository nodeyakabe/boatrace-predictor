## 優先度：高

### 1. 信頼度Bのレース数を増やす
**現状:**
- 信頼度B: 4レース (3.1%) - 的中率100%
- 信頼度C: 53レース (41.1%) - 的中率66.0%

**問題:**
- 信頼度Bが少なすぎて、高信頼度予測の機会損失
- 信頼度Cが多いが、的中率が66%と中途半端

**改善策:**
1. 信頼度判定基準の見直し
   - 現在の閾値を確認（`_calculate_confidence`メソッド）
   - Bの基準を少し緩和（例: 70点以上 → 65点以上）
   - スコアだけでなく、他要素も考慮
     - 選手の勝率
     - モーター性能
     - 会場適性

2. スコアリングロジックの改善
   - 優秀な選手への加点を増やす
   - 会場補正の効果を強化
   - 決まり手適性スコアの重み調整

**期待効果:**
- 信頼度Bを10-15%程度に増加
- 全体的な予測品質の向上

### 2. 信頼度Cの的中率向上
**現状:**
- 信頼度C: 66.0%的中（35/53）

**目標:**
- 信頼度C: 70%以上の的中率

**改善策:**
1. 会場別の補正強度調整
   - 現在: インが弱い会場は0.95、強い会場は1.05-1.10
   - より細かい調整が必要な可能性
     - 戸田: 0.95 → 0.90
     - 丸亀: 1.05 → 1.00（歴史的56.4%だが、実際44.4%の日もあった）

2. 外枠(4-6号艇)の評価改善
   - 現状: 5号艇・6号艇の予測的中率0%
   - 外枠を1着予測する時の基準強化
   - 外枠が強い条件の明確化
     - モーター性能が極めて高い
     - A1級の超一流選手
     - 会場特性（江戸川など）

3. レースグレード別の調整
   - SG/G1レースでは選手実力の重要度を上げる
   - 一般戦では会場特性・モーター性能の重要度を上げる

### 3. 会場特性のさらなる分析
**改善策:**
1. 季節・時期による変動を考慮
   - 夏季 vs 冬季
   - 天候条件

2. より詳細な会場特性
   - 2号艇・3号艇の勝率
   - 風向きの影響
   - 水質（海水 vs 淡水）
